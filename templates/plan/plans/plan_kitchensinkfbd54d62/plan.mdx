## URL Shortener — Overview

A tiny service that turns long URLs into short `/s/<code>` links and tracks clicks.

### Why it exists

Sharing raw tracking URLs is ugly and **unmemorable**. We want _short, branded_ links with basic analytics.

Core capabilities:

- Create a short code for any URL
- Redirect `/s/<code>` to the original target
- Count clicks per link

Rollout order:

1. Schema + create endpoint
2. Redirect handler
3. Click counting
4. Dashboard

Acceptance checklist:

- [x] Schema designed

- [x] Create endpoint stubbed

- [ ] Redirect handler wired

- [ ] Click analytics surfaced

> Keep codes short and collision-safe — base62 over an auto-increment id beats random retries.

```ts
function toBase62(n: number): string {
  const A = "0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ";
  let s = "";
  do {
    s = A[n % 62] + s;
    n = Math.floor(n / 62);
  } while (n > 0);
  return s;
}
```

| Field     | Type    | Notes                        |
| --------- | ------- | ---------------------------- |
| code      | text    | base62, unique, the PK       |
| targetUrl | text    | original long URL            |
| clicks    | integer | incremented on each redirect |

</RichText>

<Callout id="callout-decision" title="Callout" tone="decision">

We will generate short codes from a **base62 encoding** of the row id rather than random strings. This makes codes collision-free without a retry loop, and the redirect lookup stays a single `SELECT` on the primary key.

</Callout>
